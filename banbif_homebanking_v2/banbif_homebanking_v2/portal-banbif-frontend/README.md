# HOMEBANKING FRONTEND

Frontend del Homebanking BanBif.

Permite al cliente ingresar al sistema, consultar cuentas, revisar movimientos y visualizar informacion relacionada a sus productos financieros.

## Tecnologia

- React
- Vite
- JavaScript

## Estructura principal

- src: codigo principal del frontend.
- dist: compilacion generada.
- node_modules: dependencias locales.
- package.json: scripts del proyecto.

## Ejecucion

npm.cmd run dev

URL local:

http://localhost:5173

## Compilacion

npm.cmd run build
