# CORE FINANCIERO FRONTEND

Frontend interno del Core Financiero BanBif.

Permite revisar solicitudes de credito, evaluar reglas de negocio, aprobar, rechazar, derivar, desembolsar y gestionar recuperaciones.

## Tecnologia

- React
- Vite
- JavaScript

## Estructura principal

- src: codigo principal del frontend.
- public: recursos publicos.
- dist: compilacion generada.
- node_modules: dependencias locales.
- .env.local: configuracion local.
- package.json: scripts del proyecto.

## Ejecucion

npm.cmd run dev

URL local:

http://localhost:5174

## Compilacion

npm.cmd run build
